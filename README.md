# system_package-apps

## Shell Scripts by Shandryll

Automating Installation

Make sure the file has execute permission.

## Programs to Ubuntu
* Git
* Python pip
* NodeJS
* Docker
* Docker Compose
* Visual Studio Code
* Google Chrome
* Dropbox
* Flameshot
* Steam
* Discord
* Spotify

## Commands:

### Set execute permission on your script:
```
chmod +x ${distro}_package-apps.sh
```

### Run the code:
```
sudo sh ${distro}_package-apps.sh
```
