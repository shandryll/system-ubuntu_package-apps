# system_package-apps

## Shell Script by Shandryll

Automating Ubuntu Installation

Make sure the file has execute permission.

## Programs

* Git
* Python pip
* NodeJS
* Docker
* Docker Compose
* Visual Studio Code
* Google Chrome
* Dropbox
* Flameshot
* Steam
* Discord
* Spotify

## Commands:

### Set execute permission on your script:
```
chmod +x package_apps.sh
```

### Run the code:
```
sudo sh package_apps.sh
```
